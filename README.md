# VRTrainer Client

Trainer:
- Navigate to `Trainer` tab
- Set correct `Input Device`
- Select already existing, or create new `Pet Profile`
- Set `Names`, `Scolding Words`
- Toggle individual `Focus`, `Proximity`, `Tricks`, `Scolding`, `Ear/Tail Pull`, `Pronouns` features
- Set difficulty sliders
- Navigate to `Server` tab
- Select `Trainer` role
- Press `Start` or `Join`
- Assign a profile to a pet

Pet:
- Navigate to `Pet` tab
- Set correct `Input Device`
- Set correct `PiShock Credentials`
- Navigate to `Server` tab
- Select `Pet` role
- Press `Start` or `Join`
